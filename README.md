This repository is a starter kit for a production-grade "Seoul Hotel Map" project:
- Frontend: React + Vite + TypeScript + Tailwind (map page + admin panel skeleton)
- Backend: Node.js + Express + TypeScript (simple API + S3 presign stub)
- DB: PostgreSQL suggested (not included); sample schema in /backend/schema
- Deployment: instructions included in README

This skeleton is intended for a developer to clone and finish configuration:
1. Fill environment variables (.env)
2. `cd backend && npm install` and `cd frontend && npm install`
3. Run backend: `npm run dev` (uses ts-node-dev)
4. Run frontend: `npm run dev` (Vite)

See /backend/README.md and /frontend/README.md for details.
